Step 1 — Submission Method

How would you
like to submit?

Build on the Website

Use our character creator to build your sheet right here. You get full creative control with these tools:

  • Custom fonts & text sizes
  • Text colour palette
  • Bold & italic formatting
  • Click-to-reveal redacted text
  • Glitchy text animation effect
  • Image upload (up to 5MB)
  • Live ACS bar preview (SCP entries)
  • Multiple collapsible addenda (SCP)
Use Website Builder

Submit a Google Doc

Already have a character sheet written up in Google Docs? Just paste the link. Make sure it's set to "Anyone with the link can view" before submitting.

Staff will review the document directly. Approved Google Doc submissions open the original document when viewed on the roster.

TYPE
Step 2 — Character Type

What type of
character is this?

Foundation Personnel

Foundation researchers, Class-D personnel, MTF operatives, O5 council members. Full structured format with clearance level, classification, department, and rich text sections.

SCP Entry

Submit an anomaly using the full SCP document format — complete with a live ACS bar, containment procedures, description, collapsible addenda, and abilities.

Unaffiliated Personnel

Characters operating outside the Foundation — civilians, rogue agents, GoI members, or anyone who doesn't fit neatly into a box. Open format, more freedom.
