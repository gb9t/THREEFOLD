Step 1 of 2 — Submission Method

How would
you like
to submit?

Method/Type/Sheet
Google Doc

Submit a Google Doc

Already have a sheet written in Google Docs? Paste the link and submit. Make sure sharing is set to "Anyone with the link can view" before submitting.

Staff will review the document directly. Once approved, your character appears on the server roster.

Submit a Link
Wikidot

Submit a Wikidot Link

Prefer hosting your sheet on your own Wikidot page? Paste the public URL and submit. Make sure the page is set to publicly viewable so staff can access it without an account.

Staff will review the page directly. Once approved, your character appears on the server roster.
